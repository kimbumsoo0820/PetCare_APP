## PETCARE 앱 (PetStar) 

>게임 이름 : PetStar <br/>
>설명 : 반려동물을 기르는 세대가 점점 늘어남에 따라 이를 관리하기 위한 가이드가 필요하다고 생각하여 제작하게 되었다. <br/>
>기술: JAVA <br/>
>프레임워크 : Android Studio <br/>
